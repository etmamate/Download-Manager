# 🚂 Download Manager

Um gerenciador de downloads simples e elegante para facilitar o acesso aos instaladores dos softwares mais utilizados no dia a dia.

## 📋 Sobre o Projeto

O **Download Manager** é uma aplicação web que centraliza os links de download de programas essenciais, tornando mais rápido e organizado o processo de instalação de softwares. Com uma interface limpa e intuitiva, o usuário pode acessar os downloads com apenas um clique.

## ✨ Funcionalidades

- ✅ Interface simples e responsiva
- ✅ Botões estilizados com efeitos hover e feedback visual
- ✅ Links diretos para download dos seguintes softwares:
  - Anydesk
  - Slack
  - Google Chrome
  - Easy (com informações de senha)
  - Escolaweb
  - Sentinela
- ✅ Seção informativa com senhas necessárias para instalação/desinstalação do Easy

## 🛠️ Tecnologias Utilizadas

- **HTML5** - Estrutura da página
- **CSS3** - Estilização e responsividade
- **JavaScript** - Funcionalidade dos botões e redirecionamentos
